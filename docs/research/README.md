**Penfold baseline:** v0.2.6 — local-first Flutter notebook, Android/S Pen, SQLite, no cloud.

## How to use

1. Read each app's lists for strengths/weaknesses and recurring complaint themes.
2. Compare **Penfold Takeaway** sections across apps to find shared gaps and differentiation.
3. Feed prioritized items into [CHANGELOG.md](../../CHANGELOG.md) and GitHub Issues.
