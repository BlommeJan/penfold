## How to use

1. Read **[SYNTHESIS.md](SYNTHESIS.md)** for the consolidated picture and priority roadmap.
2. Drill into each app's file for strengths/weaknesses and recurring complaint themes.
3. Compare **Penfold Takeaway** sections across apps to find shared gaps and differentiation.
4. Feed prioritized items into [CHANGELOG.md](../../CHANGELOG.md) and GitHub Issues.
