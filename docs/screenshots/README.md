# README screenshots

Marketing and README images for Penfold live here. The main [README.md](../../README.md) gallery links to these files by relative path.

## Where to put files

Add PNG or JPG screenshots directly in this folder (`docs/screenshots/`). Do not commit placeholder images — only real captures from a device or emulator.

## Recommended filenames

Use two-digit prefixes so the gallery order stays stable:

| File | What to capture |
|------|-----------------|
| `01-library.png` | Home library: colored notebook covers, folders, tag filter chips, search bar |
| `02-notebook-editor.png` | Active notebook: handwriting on ruled/grid paper, top toolbar, floating page pill |
| `03-page-overview.png` | Page overview grid with thumbnails, reorder handles, OCR badge if visible |
| `04-page-themes.png` | Page settings or side-by-side pages showing paper themes (light, dark, sepia, pastels) |
| `05-settings.png` | Settings screen: toolbar order, data/backup, About, OCR dictionary |
| `06-dark-mode.png` | App in dark theme (library or editor) |
| `07-languages.png` | Language/locale picker showing supported translations |
| `08-ocr-search.png` | Handwriting OCR: indexed page badge and/or search results for ink |
| `09-spen-toolbar.png` | Stylus workflow: tool palette, lasso/shapes, or S Pen barrel-button erase hint |

Add `10-…`, `11-…`, etc. for extras (PDF import, trash, export sheet) if useful.

## Image tips

- **Aspect ratio:** Prefer tablet landscape (16:10 or 4:3) — Penfold targets Galaxy Tab–class devices.
- **Format:** PNG preferred for crisp UI text; JPG is fine for large galleries.
- **Resolution:** 1920×1200 or similar is enough; avoid upscaled or heavily compressed shots.
- **Content:** Use sample notebooks with readable ink, not personal data.
- **Consistency:** Same device theme (light vs dark) per row in the README table reads best.

After adding files, open the main README on GitHub to confirm paths and captions look right.
